Maternal Health Risk Executive Dashboard

An interactive corporate-grade business intelligence dashboard built to analyze clinical vitals and risk stratification across 1,014 expectant mothers.

## Live Interactive Demo
https://myhierah16.github.io/Maternal-Health-Risk-Executive-Dashboard/

## Key Insights & Metrics
* **Total Patients Tracked:** 1,014
* **High-Risk Prevalence:** 26.8% 
* **Core Risk Indicators:** Identifies strong clustering where advanced maternal age combined with elevated Systolic Blood Pressure and blood sugar correlates directly with high-risk pregnancy classifications.

## Tech Stack & Skills
* **Language:** Python
* **Libraries:** Plotly Graph Objects, Plotly Subplots, Pandas
* **UI Design:** Subplot grid mapping with a clean executive corporate theme layout.
